# ESPHome-Alarm-Clock
Alarm clock built in ESPhome with the following features:
- MAX7219 matrix display.
- Display adapts to surrounding lights.
- Display can be turned on and off by button and Home Assistant.
- Configurable alarm for every day of the week.
- Piezo buzzer based alarm.
- Silent alarm possible and let HA arrange the sound, with piezo fallback if there is no connection to HA.
- Alarm countdown timer for automations that need to be set in motion before the alarm (e.g. turning on the coffee maker).
- Alarm once setting so your not suprised in the next weekend :-)
- External alert mode with siren sound. Allows you to tie in the alarm clock with a fire or burglar alarm.
- External, internal temperature and TVOC values visible at the push of a button.
- Extensive configuration parameters in HA.
- With custom font to make full use of the display.
- Switchable led from HA to convey a message
